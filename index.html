IV. Object contructor

  1. Object contructor là gì

    Là một bản thiết kế được tạo ra để các đối tượng khác có thể kế thừa và có những thể hiện khác nhau

  2. Các bước tiến hành

    1. Khái báo một function contructor với tên function bắt đầu bằng một chữ cái in hoa (đây là một bản thiết kế)

    2. Khởi tạo các đối tượng kế thừa bản thiết kế có sẵn bằng cú pháp

      Có thể thêm các key mới vào trong các đối tượng kế thừa mà không là ảnh hưởng đến đối tượng khác
      Ví dụ

    3. Bên trong object contructor có thể là một function và các object thừa kế cũng có method này

V. Object prototype

  1. Object prototype là gì

    Tất cả các object được khởi tạo trong Javascript đều được kế thừa từ một object contructor nào đó. Các object được các object khác kế thừa gọi là object prototype

  2. prototype là gì

    Tất cả các properties bên trong object prototype được gọi là prototype object contructor nào đó. Các object được các object khác kế thừa gọi là object prototype

  3. Chứng minh những điều này

    1. Khi khởi tạo một object với cú pháp bất kỳ, tiến hành log đối tượng object này ra, ta sẽ luôn thấy một thuộc tính prototype của một object prototype có tên là Object


      Trong quá trình duyệt qua các properties của mảng ta phải chú ý đến các properties riêng của object đó với các properties chung nằm trong prototype

    2. Bây giờ ta log ra đối tượng Object, ta sẽ thấy nó là một function contructor

  4. Bây giờ ta muốn thêm một properties vào một object contructor hoặc một object prototype phải thông qua prototype của object prototype
